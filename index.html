Case Study 1 — Smart Transit Optimization

Challenge: Frequent congestion on commuter corridors.

Solution: Real-time telemetry from vehicles + adaptive signal timing reduced travel time by 18% during peak hours. Outcome: 12% drop in emissions on main corridor.

Case Study 2 — Microgrid Resilience

Challenge: Keeping critical services online during storms.

Solution: Decentralized solar + battery microgrids with islanding capability. Outcome: 99.7% uptime for hospitals and emergency centers in outages.

Case Study 3 — Predictive Waste Collection

Challenge: Inefficient routing and overflowing bins.

Solution: Bin sensors + ML predicted fill rate, enabling dynamic pickup routing. Outcome: 30% fewer collection miles and cleaner public spaces.

Case Study 4 — Water Leak Detection

Challenge: Undetected pipe leaks leading to waste.

Solution: Pressure & acoustic sensors with edge analytics flagged anomalies. Outcome: 40% faster repair times and 22% water savings.

Case Study 5 — Air Quality Zoning

Challenge: Localized pollution hotspots near industry.

Solution: Dense sensor network + targeted green buffers reduced peak PM2.5 events. Outcome: Public alerting helped at-risk groups avoid exposure.

Case Study 6 — Smart Parking & EV Charging

Challenge: Low EV charging utilization, parking mess.

Solution: Integrated parking/charger reservation system with demand forecasting. Outcome: 85% utilization at peak and improved turnover.

Case Study 7 — Public Safety Analytics

Challenge: Slow response times in certain districts.

Solution: Predictive dispatching and optimized patrol routing using historical incident data. Outcome: 15% faster average emergency response time.

Case Study 8 — Citizen Data Portal

Challenge: Low civic engagement and transparency.

Solution: Open dashboards + mobile reporting app increased participation in local planning. Outcome: 25% rise in feedback submissions and faster issue resolution.

Case Study 9 — Heat Island Mitigation

Challenge: Urban heat islands in dense commercial zones.

Solution: Cool-roof program, reflective pavements, and additional tree canopy. Outcome: Avg ambient temperatures dropped 1.8°C in treated zones.

Case Study 10 — Logistics & Last-Mile Delivery

Challenge: Peak congestion from delivery fleets.

Solution: Consolidation hubs + off-peak delivery incentives reduced midday truck traffic. Outcome: Delivery times improved and congestion eased.

Show: Dijkstra's Algorithm (C++)

Toggle the code to view a clean, ready-to-compile implementation of Dijkstra's shortest path algorithm (useful for routing & map pathfinding). 

#include <bits/stdc++.h>
using namespace std;

// Dijkstra's algorithm using a min-priority queue.
// Graph is represented as adjacency list: vector<vector<pair<int,int>>> adj;
// Each pair = (neighbor, weight)

const long long INF = (1LL<<60);

vector<long long> dijkstra(int n, int src, const vector<vector<pair<int,int>>>& adj) {
    vector<long long> dist(n + 1, INF); // 1-based nodes; change if 0-based
    dist[src] = 0;
    // min-heap of pairs (distance, node)
    priority_queue<pair<long long,int>, vector<pair<long long,int>>, greater<pair<long long,int>>> pq;
    pq.push({0, src});

    while (!pq.empty()) {
        auto [d, u] = pq.top(); pq.pop();
        if (d != dist[u]) continue; // stale entry
        for (auto &edge : adj[u]) {
            int v = edge.first;
            long long w = edge.second;
            if (dist[v] > d + w) {
                dist[v] = d + w;
                pq.push({dist[v], v});
            }
        }
    }
    return dist;
}
